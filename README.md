## Getting Started

### Prerequisites
- Python 3.x
- Required dependencies (see requirements.txt)

### Installation

1. Clone the repository
```bash
git clone https://github.com/Varshil-0101/OOC-Neural--Architect.git
cd OOC-Neural--Architect
```

2. Install dependencies
```bash
pip install -r requirements.txt
```
